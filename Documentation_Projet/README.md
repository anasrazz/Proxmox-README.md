# Documentation et Suivi du Projet

Ce dossier centralise l'ensemble des documents administratifs et de suivi transmis à l'encadrant durant les différentes phases du projet PFE-HP1 2026.

## 📂 Contenu
- **Fiches de cadrage** : Documents définissant les objectifs et le périmètre.
- **Comptes-rendus** : Rapports d'avancement hebdomadaires/mensuels.
- **Rapports intermédiaires** : État d'avancement technique et blocages rencontrés.
- **Documents administratifs** : Correspondances et documents validés par l'encadrant.

## 📅 État de validation
Chaque document est indexé par date pour assurer un suivi chronologique clair.
