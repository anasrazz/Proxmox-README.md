# Supervision avec Grafana & Prometheus

Ce dossier contient les éléments nécessaires à la supervision de notre infrastructure virtualisée.

## 📝 Résumé des actions
- **Collecte** : Mise en place de Prometheus pour la collecte des métriques (Node Exporter).
- **Visualisation** : Création de tableaux de bord Grafana pour monitorer l'état des machines virtuelles et des ressources Proxmox.
- **Alerting** : Configuration des seuils d'alerte sur les ressources critiques (CPU, RAM, Stockage).

📄 **Détails techniques** : Consultez le document [grafana_report.pdf](./grafana_report.pdf) pour voir les dashboards et les configurations.
